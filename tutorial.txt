Тутор по гиту


что бы привязать git к проекту нужно написать: git init
что бы добавить файл к проекту нужно написать: git add "название файла" или "." что бы добавить все
что бы посмотреть статус файлов нужно написать: git status
что бы удалить файл из проекта нужно написать: git rm --cached "название файла"
что бы не записывать историю файлов нужно создать папку: .gitignore и записывать туда то что нужно игнорировать

что бы посмотреть ветку в которой мы находимся нужно написать: git branch
что бы добавить новую ветку нужно написать: git branch "имя ветки"
что бы удалить ветку нужно написать: git branch -D "имя ветки"
что бы выбрать ветку нужно написать: git checkout "имя ветки"
что бы создать и сразу переместиться в новую ветку нужно написать: git checkout -b"имя ветки"
что бы совместить ветки нужно написать: git merge "имя ветки"

Работа с GitHub
что бы посмотреть имя нужно написать: git config --global user.name
что бы поменять имя нужно написать: git config --global user.name "имя"
что бы посмотреть email нужно написать: git config --global user.email
что бы поменять email нужно написать: git config --global user.email "имя"
что бы привязать репозиторий(папка с проектом) к GitHub нужно написать: git remote add origin "ссылка репозитория на GitHub"
что бы загрузить или обновить репозиторий нужно написать: git push -u origin master или main
что бы клонировать(загрузить) себе проект на комп нужно написать: git clone "ссылка репозитория на GitHub"
что бы обновить у себя проект на компе нужно написать: git pull
